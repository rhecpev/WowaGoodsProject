# WuwaGoods (WowaGoods)

> ⚠️ 앱 이름 오타 안내: 개발 중 오타가 발생하여 앱 내부명이 `WowaGoods` 로 되어있습니다. 정확한 이름은 `WuwaGoods` 입니다.

게임 명조의 굿즈 체크리스트 앱입니다.

## 기능

- 공식 굿즈 목록 조회 (시리즈별, 캐릭터별)
- 2차창작 굿즈 등록 및 관리
- 보유 여부 체크
- 캐릭터 선호 설정
- 다크/라이트 테마 지원
- 유저 데이터 백업 및 복원
- GitHub 연동 자동 업데이트

## 설치

APK 파일을 다운로드 후 설치해주세요.
알 수 없는 출처 앱 설치 허용이 필요합니다.

## 데이터 업데이트

마이페이지 → DB 업데이트에서 캐릭터, 시리즈, 굿즈 데이터를 최신으로 업데이트할 수 있습니다.

## 백업 및 복원

마이페이지 → 유저 데이터에서 보유 굿즈 데이터와 2차창작 굿즈를 ZIP 파일로 백업/복원할 수 있습니다.
